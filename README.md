# team-swifters
Студенческий проект: team-swifters
название проекта: CarShop — iOS-приложение с каталогом спортивных автомобилей.
выбранные стек и платформа: 
- iOS
- Swift
- Xcode
состав команды: 
Аленина Мария Алексеевна IOS разработчик
Ли Елисей Владиславович IOS разработчик
Петров Даниил Игоревич IOS разработчик

реализованнный функционал : 
- Просмотр каталога автомобилей (отображение фотографий, описания и стоимости автомобилей)
- Изменение количества автомобиля Aston Martin
- Просмотр подробной информации о Bugatti Chiron
- Отображение доступных цветов автомобиля
- Интерфейс разделов меню и корзины
